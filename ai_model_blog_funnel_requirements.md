# AI Model Website Requirement Document

## Project Overview
**Goal:** Build a safe, high-converting website funnel for Instagram traffic.

---

## 1. Core Objective
Create a mobile-first website that:
- Looks like a real creator/AI brand
- Avoids Instagram restrictions
- Converts traffic into clicks on "exclusive content"

---

## 2. Site Structure

### Home Page (/)
- Hero section (image + name + tagline)
- Short bio (2–3 lines)
- CTA buttons:
  - View Gallery
  - Exclusive Content 🔒
- Footer with social links

### Gallery Page (/gallery)
- Grid layout
- Lightbox preview
- Only safe (non-explicit) images

### VIP Page (/vip)
- Teaser text
- Blurred/locked previews
- CTA: Unlock Access
- Redirect to monetization link

### Blog Page (/blog)
- Articles for authenticity
- Example topics:
  - Daily life
  - Behind the scenes
  - Style & shoots

### About Page (/about)
- Short story
- Branding/personality

---

## 3. Design Requirements

- Clean, modern layout
- Mobile-first design
- Fast loading (<2s)
- Minimal clutter

---

## 4. Safety Rules (Critical)

### Must Follow
- No explicit adult words on homepage
- No direct monetization link on homepage
- No auto redirects
- No aggressive popups

### Replace Words
| Avoid | Use |
|------|-----|
| OnlyFans | Exclusive |
| 18+ | Private |
| NSFW | VIP |
| Adult | Premium |

---

## 5. Funnel Flow

Instagram → Home → VIP Page → Monetization Link

---

## 6. Functional Requirements

- Responsive design
- Fast loading
- SEO-friendly
- Basic analytics tracking

---

## 7. Conversion Strategy

### CTA Examples
